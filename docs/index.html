Alles, was Privatvermieter brauchen

Von der Mietvertragsverwaltung bis zur fertigen PDF-Abrechnung — komplett in einer Anwendung.

🧾

Nebenkostenabrechnung

Strom, Gas, Kaltwasser, Warmwasser, Heizkosten und Betriebskosten — jede Kostenart mit eigenem Abrechnungszeitraum und korrekter Tagesgenauigkeit. Fertiges PDF auf Knopfdruck.

📊

Zählerstände verwalten

Strom-, Gas-, Heizungs- und Wasserzähler (Kalt & Warm) registrieren und Ablesungen mit Datum erfassen. Verbrauch, Tagesschnitt und Verlaufsgrafik auf einen Blick.

🔐

Kautionsverwaltung

Kaution erfassen, Teilabzüge dokumentieren (NK-Nachzahlung, Schäden, Reinigung, Mietrückstand) und Restbetrag zurückbuchen — mit vollständigem Verlauf pro Vertrag.

💰

Miet- & Zahlungsverfolgung

Monatliche Zahlungen erfassen, Rückstände erkennen, Kontoauszug pro Mieter. Automatische Warnmeldungen bei auslaufenden Verträgen.

📬

Mahnung Generator

Formelle Zahlungsaufforderung als PDF generieren — mit geschlechtergerechter Anrede, Anschrift aller Mitmieter und eingebetteter Unterschrift.

📋

Mitmieter & WG-Support

Mitmieter pro Vertrag verwalten — im Vertrag aufgeführte Personen erscheinen automatisch in Anschrift und Anrede aller PDFs. Personenanzahl für die Abrechnung wird automatisch ermittelt.

🏢

Betriebskosten & Hausgeld

Wiederkehrende und einmalige Kosten pro Wohnung (Hausgeld, Grundsteuer, Versicherung, Hypothek) erfassen und im Jahresüberblick auswerten.

📈

Bilanz & Rendite

Monats- und Jahresübersicht: Soll-Mieteinnahmen, tatsächliche Zahlungen, Kosten und Netto pro Wohnung. Automatische Hinweise bei Leerstand, Mietrückstand oder negativer Rendite.

💾

Abrechnungsprofile

Einmal eingegebene Abrechnungsdaten als Profil speichern und im nächsten Jahr laden — inklusive automatischer Vertragsauswahl. Kein erneutes Eintippen von Tarifen und Zählerständen.

Nebenkostenabrechnung —
korrekt auf den Tag genau

Viele private Vermieter berechnen die Nebenkosten noch mit Excel und riskieren dabei Fehler bei Einzug, Auszug oder Mieterwechsel. Diese Software berechnet alles automatisch und tagesgenau — mit korrekter Proratierung für jeden Mieter.

  • ✓ Eigener Abrechnungszeitraum pro Kostenart (Anbieter ≠ Mietjahr)
  • ✓ Effektiver Mietzeitraum wird automatisch aus dem Vertragsdatum ermittelt
  • ✓ Zählernummern aus der Meterverwaltung werden direkt übernommen
  • ✓ Warmwasser separat abrechenbar (Frischwasser + Abwasser + Heizenergie)
  • ✓ Heizkostenverteiler (ISTA) mit Umrechnungsfaktor Einheiten → kWh
  • ✓ Abrechnungsprofile: Werte einmal eingeben, nächstes Jahr laden
  • ✓ Fertiges A4-PDF mit Anschrift, Anrede, Berechnungstabellen und Unterschrift

In 2 Schritten startklar

Sie brauchen nur Docker Desktop und Python 3.10+.

1
Repository klonen
git clone https://github.com/slgao/landlord_system.git
cd landlord_system
2
Setup-Skript ausführen
./setup.sh

Richtet automatisch ein: virtual environment, Abhängigkeiten, PostgreSQL-Container, .env und Datenbankschema.

✓
Anwendung starten
source venv/bin/activate
honcho start

Startet Streamlit und FastAPI gleichzeitig.
UI → http://localhost:8501  |  API → http://localhost:8000
